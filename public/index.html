worktree.baseRef — 워크트리 분기 기준 선택

fresh는 원격 기본 브랜치에서, head는 로컬 HEAD에서 브랜치를 딴다. 이제 워크트리 생성 기준을 직접 고를 수 있음.

NEWWorktree2026.05.07 · v2.1.133
›

설정 방법

  • worktree.baseRef: "fresh" — 항상 origin/<기본 브랜치>에서 새 워크트리를 만듦 (기존 기본값)
  • worktree.baseRef: "head" — 현재 로컬 HEAD 기준으로 브랜치를 딴다
  • --worktree 플래그, EnterWorktree 훅, 에이전트 격리 워크트리 모두에 적용됨
  • 로컬에서 직접 커밋을 쌓고 있는 상태라면 "head"로 두면 그 위에서 작업 가능

NEW명령어프로젝트 관리2026.5.1 · v2.1.126
›

사용법

Bedrock 서비스 티어 선택 — AWS 요청 우선순위 직접 지정

환경변수 하나로 Bedrock 호출 시 default·flex·priority 중 원하는 티어를 골라서 보낼 수 있음.

NEWAWSBedrock2026.4.29 · v2.1.122
›

이게 왜 필요해?

AWS Bedrock은 요청 처리 우선순위를 티어로 관리함. 기존엔 기본값만 쓸 수 있었는데, 이제 ANTHROPIC_BEDROCK_SERVICE_TIER를 설정하면 X-Amzn-Bedrock-Service-Tier 헤더로 자동 전달됨.

  • default — 기본 처리 (변경 없음)
  • flex — 유연한 처리 (비용 최적화)
  • priority — 우선 처리 (속도 우선, 엔터프라이즈용)

설정법

# .env 또는 셸 설정에 추가
ANTHROPIC_BEDROCK_SERVICE_TIER=priority

# 또는 실행 시 인라인으로
ANTHROPIC_BEDROCK_SERVICE_TIER=flex claude

prUrlTemplate + CLAUDE_CODE_HIDE_CWD — PR 배지 커스텀 & 시작 화면 정리

PR 배지를 GitHub 말고 사내 코드리뷰 URL로 바꾸고, 시작 로고에서 작업 디렉터리 경로를 숨길 수 있음.

NEW설정2026.4.24 · v2.1.119
›

prUrlTemplate — PR 배지 링크 바꾸기

기본적으로 하단 PR 배지는 github.com 링크를 가리키는데, 사내 GitLab이나 Gerrit 같은 커스텀 코드리뷰 시스템을 쓴다면 prUrlTemplate으로 URL 패턴을 직접 지정할 수 있어.

설정 방법 (settings.json)

# .claude/settings.json 또는 ~/.claude/settings.json
{
  "prUrlTemplate": "https://gitlab.mycompany.com/{org}/{repo}/-/merge_requests/{number}"
}

CLAUDE_CODE_HIDE_CWD — 시작 로고에서 경로 숨기기

Claude Code를 실행하면 시작 로고에 현재 작업 디렉터리가 표시되는데, 화면 공유하거나 스크린샷 찍을 때 경로가 노출되기 싫다면 이 환경변수로 숨길 수 있어.

# 터미널에서 바로 적용
CLAUDE_CODE_HIDE_CWD=1 claude

# 또는 쉘 설정에 영구 등록 (~/.zshrc, ~/.bashrc)
export CLAUDE_CODE_HIDE_CWD=1

Vim Visual Mode — 터미널에서 진짜 vim처럼 선택하기

이제 Claude Code 입력창에서 v, V로 텍스트 선택하고 vim 연산자까지 쓸 수 있음. vim 유저라면 손이 그대로 따라가는 느낌.

NEW에디터자주쓰임2026. 4. 23. · v2.1.118
›

뭐가 달라졌냐면

  • v — 문자 단위 Visual Mode 진입. 커서 이동하면 범위가 하이라이트됨
  • V — 줄 단위 Visual-Line Mode 진입. 통째로 한 줄씩 선택
  • 선택 후 d, y, c 같은 vim 연산자 그대로 사용 가능
  • 선택 영역이 시각적으로 바로 표시돼서 뭘 선택했는지 한눈에 보임

사용법

# Claude Code 입력창에서 vim 모드 활성화 상태일 때
v          # 문자 단위 visual mode 진입
V          # 줄 단위 visual-line mode 진입
hjkl       # 이동하면서 선택 범위 확장
d / y / c  # 삭제 / 복사 / 교체

DISABLE_UPDATES — 업데이트 완전 봉쇄

  • 기존 DISABLE_AUTOUPDATER는 자동 업데이트만 막음. 근데 claude update는 여전히 됐음
  • DISABLE_UPDATES=1 설정하면 수동 업데이트 명령어까지 전부 차단
  • CI 환경이나 버전 고정이 필요한 팀에서 유용함
